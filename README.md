# Hotel Shree Balaji - Next.js project (ready for Vercel)

This is a small Next.js + Tailwind starter containing the Hotel Shree Balaji landing page.
Files included:
- package.json
- next.config.js
- postcss.config.js
- tailwind.config.js
- pages/_app.js
- pages/index.js
- styles/globals.css

How to deploy:
1. Push this repository to GitHub.
2. Connect the repo to Vercel and deploy (Vercel auto-detects Next.js).
3. Ensure you run `npm install` (Vercel does this automatically) and then build.

Notes:
- Replace phone number placeholders (9198XXXXXXXX) with the hotel's WhatsApp number.
- All images are hotlinked to Unsplash/Postimg; you can replace them by uploading to /public.
